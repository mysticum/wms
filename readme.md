# WMS Prototype

Is a simple prototype of application for managing warehouses.

## Installation

```bash
git clone https://github.com/klementinko/wms.git
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cd wmsprototype
python manage.py runserver
```
